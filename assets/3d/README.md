# Sony A7 3D model

Do této složky nahrajte licencovaný model ve formátu GLB s přesným názvem:

`sony-a7.glb`

Výsledná cesta musí být:

`/assets/3d/sony-a7.glb`

## Doporučené parametry

- formát GLB 2.0,
- model Sony Alpha A7, A7 III, A7 IV nebo A7S III,
- maximálně přibližně 10–15 MB,
- ideálně textury 2K,
- PBR materiály,
- komprimované textury WebP nebo KTX2 jsou výhodou,
- před použitím ověřte licenci modelu pro komerční web.

Scéna model automaticky vystředí a normalizuje jeho velikost. Pokud soubor chybí nebo se nepodaří načíst, web použije vestavěný procedurální A7-style fotoaparát.

Model není součástí repozitáře, dokud není dodán soubor s odpovídající licencí.
